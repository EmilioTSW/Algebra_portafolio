Recordatorio de las fórmulas

Los productos notables que se animan en esta actividad son:

  • (a + b)² = a² + 2ab + b²
  • (a - b)² = a² - 2ab + b²
  • (a + b)(a - b) = a² - b²

Cada animación puede mostrar, por ejemplo, la distribución de términos, el desarrollo de los paréntesis y el reagrupamiento final. De esta forma, se conecta la fórmula memorizada con el proceso algebraico que la justifica.

Tip: aunque aquí se usa notación simbólica, puedes relacionar luego estos patrones con valores numéricos como hiciste en la Actividad 6 para reforzar la comprensión.
