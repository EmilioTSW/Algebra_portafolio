Conversión de la forma polar a la forma exponencial

Ingresa el módulo r y el ángulo θ (en grados) para ver cómo se representa el número complejo en forma polar y en forma exponencial r·eiθ. Esta actividad te ayuda a relacionar la notación geométrica con la notación exponencial de Euler.

Datos del número complejo
Esperando datos… Ingresa un valor para r y θ y luego haz clic en Convertir.
Recordatorio teórico

Un número complejo en forma polar se escribe como:

z = r·(cos θ + i·sen θ)

Usando la fórmula de Euler, la forma exponencial es:

z = r·eiθ

  • r es el módulo o longitud del vector complejo.
  • θ es el ángulo de giro medido desde el eje real positivo.
  • Para trabajar en radianes, se usa: θ(rad) = θ(grados) · π / 180.
Tip: prueba con valores como r = 1, θ = 0, r = 1, θ = 90 o r = 2, θ = 45 para ver cómo cambia la expresión exponencial y la interpretación geométrica.
