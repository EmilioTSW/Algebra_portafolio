Recordatorio de productos notables

Algunas fórmulas básicas que se usan en esta actividad:

  • (a + b)² = a² + 2ab + b²
  • (a - b)² = a² - 2ab + b²
  • (a + b)(a - b) = a² - b²

Cambiando los valores de a y b, puedes observar cómo estas fórmulas generan distintos resultados numéricos, pero siempre respetan el patrón algebraico del producto notable.

Tip: prueba valores simples como a = 1, b = 1, a = 2, b = 3 o incluso números negativos, por ejemplo a = -2, b = 4, para comparar los resultados.
