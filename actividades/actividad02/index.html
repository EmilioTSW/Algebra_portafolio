Operaciones con a + bi y c + di

En esta actividad podrás ingresar dos números complejos en forma a + bi y c + di, realizar operaciones básicas y observar sus resultados. Esto permite explorar propiedades como la conmutatividad y asociatividad de la suma y la multiplicación de números complejos.

Propiedades a explorar

Puedes usar la calculadora para verificar algunas propiedades de los números complejos:

  • Conmutatividad de la suma: a + b = b + a
  • Conmutatividad del producto: a · b = b · a
  • Asociatividad de la suma: (a + b) + c = a + (b + c)
  • Asociatividad del producto: (a · b) · c = a · (b · c)
  • Conjugado: Para z = a + bi, su conjugado es z̄ = a − bi
Tip: puedes probar con valores sencillos como 1+2i, 3-i, -2+4i, etc., y comparar manualmente los resultados para comprobar las propiedades.
