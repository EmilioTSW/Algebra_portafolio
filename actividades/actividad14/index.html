Resolución de ecuaciones de primer grado de la forma ax + b = c

Introduce los coeficientes a, b y c de la ecuación ax + b = c y observa el procedimiento para despejar x. Se mostrará la ecuación original, las operaciones realizadas sobre ambos lados y el valor final de la solución.

Datos de la ecuación
Esperando datos… Ingresa a, b y c, y luego haz clic en Resolver ecuación.
Procedimiento general

Para una ecuación de la forma ax + b = c, el objetivo es despejar x. El procedimiento algebraico estándar es:

  1. Restar b en ambos lados:
    ax + b − b = c − b
  2. Simplificar:
    ax = c − b
  3. Dividir entre a (si a ≠ 0):
    x = (c − b) / a

Si a = 0, la ecuación ya no es de primer grado: puede no tener solución o tener infinitas soluciones dependiendo de los valores de b y c.

Tip: prueba con ecuaciones como 2x + 3 = 11, -3x + 5 = 2 o 0x + 4 = 4 para ver distintos tipos de soluciones (única, imposible, indeterminada).
