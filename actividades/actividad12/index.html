Factorización de trinomios de la forma x² + bx + c

Introduce los coeficientes b y c del trinomio x² + bx + c y observa el proceso de factorización animado: se buscarán dos números que multiplicados den c y sumados den b. Si la factorización es posible con enteros, se mostrará el resultado en forma de producto de binomios.

Datos del trinomio
Esperando datos… Ingresa b y c, y luego haz clic en Factorizar.
Idea de la factorización

Para factorizar un trinomio de la forma x² + bx + c buscamos dos números enteros m y n tales que:

  • m + n = b
  • m · n = c

Si existen esos números, entonces podemos escribir:

x² + bx + c = (x + m)(x + n)

Si no encontramos tales enteros m y n, el trinomio no es factorizable mediante este método simple (al menos usando solo números enteros).

Tip: prueba casos como b = 5, c = 6 (x² + 5x + 6), b = 1, c = -6 (x² + x - 6) o b = -7, c = 10 (x² - 7x + 10) y observa cómo se encuentran los factores.
